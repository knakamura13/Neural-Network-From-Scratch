# Neural Network From Scratch

A personal project that will be used to follow along with Sentdex's course on how to create a neural network from scratch using Python.

### Pre-requisites
 - Python 3.7.7

### Dependencies
 - matplotlib==3.2.1
 - numpy==1.18.2
